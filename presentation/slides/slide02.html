Problem Statement

Given: n items, each with value vi and weight wi, and a knapsack of capacity W.

Find: a subset S ⊆ {1, ..., n} that

   maximises  Σ vi · xi  subject to  Σ wi · xi ≤ W,  xi ∈ {0, 1}

  • 0/1 means each item is either taken whole or skipped — no fractions.
  • NP-hard, but admits a pseudo-polynomial O(nW) DP because W is bounded.
  • The problem has been studied since 1957 (Dantzig); still actively used today.
