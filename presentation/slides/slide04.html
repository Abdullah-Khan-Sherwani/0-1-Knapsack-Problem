Why These 6 Algorithms?

Exact (4)

Always return the true optimum. Differ in how they search and memory needed.

  • Brute Force — recursion, no caching
  • Memoization — top-down DP, cached
  • Tabulation — bottom-up DP, recoverable items
  • Space-Optimised DP — O(W) memory

Approximation (2)

Trade a bounded amount of accuracy for huge speed/memory savings.

  • Greedy — value/weight ratio, ≥ OPT/2
  • FPTAS — scaled DP, ≥ (1−ε)·OPT, tunable

Three axes to trade off: time ↔ memory ↔ accuracy. Picking the right algorithm depends on which axis is the binding constraint for your input.
